This is our Project File Repository.

To be able to Run this project, you must first install django and python:
1.Required Python version to be installed is 3.9.6 you can install it by going in this site https://www.python.org/downloads/release/python-396/
2.After that you need to install django using pip install Django.

Packages needed to be installed:
-pip install Pillow
-pip install django-filter
-pip install pytz
-pip install pisa
-pip install xhtml2pdf

After installing everything you can open our project in VSCode or IDE of your choice and run:
1. python manage.py makemigrations
2. python manage.py migrate
3. python manage.py runserver
